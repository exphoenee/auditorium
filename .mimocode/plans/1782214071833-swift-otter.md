# domelemjs npm csomag migráció

## Cél
A helyi `file:../DOMelemJS` referenciát lecserélni az npm registry-beli `domelemjs` csomagra, és eltávolítani a helyi másolatot.

## Jelenlegi állapot
- A `package.json`-ban már szerepel `"domelemjs": "file:../DOMelemJS"` — ezt kell lecserélni `npm i domelemjs` telepítésre
- Az összes komponens importja már `'domelemjs'`-t használ (Auditorium.js, Sector.js, Seat.js, Row.js)
- A tag értékek már string literálok (`"div"`, `"p"` stb.) — nincs változó alapú hivatkozás
- `src/utils/domelemjs/` helyi másolat még létezik
- A teszt mock útvonala még a régi helyi útvonalra mutat

## Teendők

### 1. Telepítés az npm registry-ből
```bash
cd E:\Projects\auditorium
npm uninstall domelemjs
npm i domelemjs
```

### 2. Teszt mock frissítése
**Fájl:** `src/__tests__/Auditorium.test.js:4`
```js
// Régi:
vi.mock('../../utils/domelemjs/domelemjs.js', () => ({
// Új:
vi.mock('domelemjs', () => ({
```
A mock callback-ben a `div`, `input`, `button`, `p` exportok eltávolíthatók, mert az npm csomag nem exportálja őket és a komponensek sem használják.

### 3. Helyi másolat törlése
```bash
rm -rf src/utils/domelemjs
```

## Ellenőrzés
- `npm run build` — győződjünk meg róla, hogy a build sikeres
- `npm run test` — futtassuk le a teszteket
